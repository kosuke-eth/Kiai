# KIAI Demo Video Script

This is a `2 minute 15 second` screen-recorded demo script for remote viewers.

It is written for async judging and review, so it includes:

- clear problem framing
- explicit hackathon context
- smooth screen transitions
- concise technical credibility
- a brief admin/operator segment

Use this flow:

1. `/`
2. `/predictions`
3. `/leaderboard`
4. `/admin`
5. strongest final screen for the close

## Recording Notes

Before recording:

- make sure at least one active scenario is visible in `/predictions`
- make sure badge claim and energy claim work in the current browser session
- use `50` energy for the allocation step
- if you plan to click an admin action, only use one you have already tested
- keep mouse movement slow and deliberate
- pause briefly after each major click so remote viewers can follow the flow

Narration style:

- calm
- precise
- slightly slower than normal conversation
- confident, but not hype-heavy
